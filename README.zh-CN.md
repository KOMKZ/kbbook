## 📲 构建 Android APK

```bash
# 第一步：检查机器环境
bash scripts/check.sh --apk

# 第二步：构建 + 安装到平板
make verify-app
```

**`make verify-app` 做了什么**：清理构建 → Capacitor 同步 → 修复包名 → 自动升版本号 → Gradle 编译 → 验证 APK 内容 → adb 安装 → 启动验证。

**前置条件**：Java JDK 11+、Android SDK（设置 `ANDROID_HOME`）、adb。先跑 `bash scripts/check.sh --apk` 验证。
